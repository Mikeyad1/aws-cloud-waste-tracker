# Policy documents for reference

This repository includes example Terms of Use and Privacy Policy text for Cloud Waste Tracker. Review these documents with qualified counsel and adapt jurisdiction-specific details before publishing.

The application’s technical assumptions for data handling and credentials are documented in: [../PRIVACY_AND_SECURITY.md](../PRIVACY_AND_SECURITY.md).

## In-app policy links

If you publish Terms/Privacy at public URLs, you can link them from the Streamlit UI using:

- `CWT_TERMS_URL`: public URL to your published Terms of Use
- `CWT_PRIVACY_URL`: public URL to your published Privacy Policy

If these variables are not set, the UI shows short informational summaries in-app.
